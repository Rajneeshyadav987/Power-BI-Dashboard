# Healthcare Analytics Dashboard

## 📌 Project Overview

The **Healthcare Analytics Dashboard** is a data analytics project developed using **Power BI** to analyze hospital data, patient admissions, billing amounts, blood types, and medical test results.

This interactive dashboard helps visualize **patient admission types, billing amount trends, test result distributions, and healthcare resource utilization**.

The project demonstrates how **data visualization and business intelligence tools can support healthcare decision-making and hospital management**.

---

## 🎯 Project Objectives

* Analyze **hospital billing amounts and revenue trends**
* Study **patient admission types (Elective, Urgent, Emergency)**
* Examine **medical test result distributions**
* Analyze **blood type and test result relationships**
* Identify **doctor and patient information insights**
* Visualize **hospital resource usage such as room allocation**

---

## 📈 Dashboard Insights

### 1️⃣ Patient Admission Types

The donut chart shows the **distribution of hospital admissions**.

Admission categories include:

* Elective
* Urgent
* Emergency

**Key Insight**

* Patient admissions are relatively balanced across different admission types, indicating varied healthcare service demands.

The bar chart analyzes **billing amounts based on medical test results**.
